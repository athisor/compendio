# Interface Zero 3.0 - SWADE Compendiums

Compendios compartidos de Interface Zero 3.0 para Savage Worlds Adventure Edition en Foundry VTT V13+.

## Instalación

### Desde Foundry VTT

1. Abre Foundry VTT
2. Ve a **Configuración** → **Add-on Modules** → **Install Module**
3. Pega esta URL en el campo "Manifest URL":
   ```
   https://raw.githubusercontent.com/TU_USUARIO/swade-setting-iz3/main/module.json
   ```
4. Haz clic en **Install**
5. Activa el módulo en tu mundo de juego

### Requisitos

- Foundry VTT V13.0 o superior
- Sistema SWADE 4.0.0 o superior

## Contenido

Este módulo incluye los siguientes compendios:

- **IZ3 - Reglas**: Documentación y reglas de Interface Zero 3.0
- **IZ3 - Ascendencias**: Ascendencias disponibles en IZ3
- **IZ3 - Orígenes**: Orígenes de personaje
- **IZ3 - Ventajas**: Edges específicos de IZ3
- **IZ3 - Desventajas**: Hindrances específicos de IZ3
- **IZ3 - Equipo**: Equipamiento y tecnología
- **IZ3 - Hacking**: Reglas y elementos de hacking
- **IZ3 - Bestiario**: NPCs y criaturas
